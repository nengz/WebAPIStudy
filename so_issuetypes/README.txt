
This folder contains the types of web API issues identified from 1885 sampled SO questions relevant to web APIs.

1. qnumrange_qids.json: the five ranges divided by the number of SO questions retrieved for web APIs, as well as the sample size calculated for each range.
2. qnumrange_sampledqids.json: the set of sampled SO question ids for each of the five ranges.
3. qnumrange_statis.xlsx: the statistics of SO questions in the five ranges, as listed in Table 7 of the paper.
4. question_issuetypes.xlsx: the manually identified web API issue types from each of the 1885 sampled SO questions.
5. qnumrange_issuetypes.json: the statistics of the identified issue types belonging to each of the five ranges.
